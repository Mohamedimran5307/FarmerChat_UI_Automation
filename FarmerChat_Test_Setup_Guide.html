Maestro UI Test Suite — Local Setup & Run Guide

Version 2.2 • June 2026

Overview

This guide provides step-by-step instructions to set up and run the FarmerChat automated UI test suite on your local machine using the Maestro framework.

✓

What You'll Achieve

Run 5 automated test cases covering core FarmerChat functionality including location services, AI chat, and user authentication.

⚡ Quick Start (Just 2 Commands!)

For experienced users, run these two commands to get started immediately:

📦 Command 1: Setup (one-time only)

brew install android-platform-tools && brew install --cask temurin@21 && curl -Ls "https://get.maestro.mobile.dev" | bash && git clone https://github.com/digitalgreenorg/farmerchat-functional-automation.git && cd farmerchat-functional-automation

Installs ADB + Eclipse Temurin JDK 21 (required by Maestro) + Maestro CLI, then clones the repo. The Temurin cask auto-registers with macOS so java is on PATH immediately — no JAVA_HOME wrangling needed.

▶️ Command 2: Run Tests

./run_tests.sh "Your Name"
⚠️

Before Running

Make sure: (1) Android device connected via USB, (2) USB Debugging enabled, (3) FarmerChat app installed. FarmerChat is now globally released, so no GPS mocking is required — the app uses the device's real location.

1 Install ADB (Android Debug Bridge)

🍎 macOS

$ brew install android-platform-tools

🐧 Linux (Ubuntu/Debian)

$ sudo apt update $ sudo apt install adb
💡

Verify Installation

Run adb version — you should see output like: Android Debug Bridge version 1.0.41

2 Install Java JDK + Maestro CLI

Maestro is a JVM-based CLI, so a Java JDK (17 or 21) must be on PATH before installing or running Maestro. Skipping this is the #1 reason Maestro fails to start.

2a. Install Java (JDK 21 LTS)

🍎 macOS

$ brew install --cask temurin@21

The Temurin cask auto-registers with macOS — java is on PATH immediately, no JAVA_HOME setup needed.

🐧 Linux (Ubuntu/Debian)

$ sudo apt update $ sudo apt install openjdk-21-jdk
💡

Verify Java

Run java -version. You should see openjdk version "21..." (or 17...). If it says "command not found," fix PATH before continuing — Maestro will not start without Java.

2b. Install Maestro CLI

Run the following command on macOS or Linux:

$ curl -Ls "https://get.maestro.mobile.dev" | bash
⚠️

Add to PATH (if needed)

If the maestro command is not found, add this line to your ~/.zshrc or ~/.bashrc:

export PATH="$PATH:$HOME/.maestro/bin"
💡

Verify Installation

Run maestro --version to confirm successful installation.

3 Enable USB Debugging on Android Device

Follow these steps on your Android device:

  1. Go to Settings → About Phone
  2. Tap Build Number 7 times to enable Developer Options
  3. Go back to Settings → Developer Options
  4. Enable USB Debugging toggle
  5. Connect device to computer via USB cable
  6. Tap Allow when prompted for USB debugging permission on device
💡

Verify Device Connection

Run the command below. Your device should appear in the list:

$ adb devices # Expected output: List of devices attached 1a0a08f0 device

4 Install FarmerChat APK

Ensure the FarmerChat app is installed on your device.

Property Value
Package Name org.digitalgreen.farmer.chat
Install Command adb install path/to/farmerchat.apk
⚠️

APK Not Available?

Contact your team lead to obtain the latest FarmerChat APK file.

5 Clone the Test Repository

$ git clone https://github.com/digitalgreenorg/farmerchat-functional-automation.git $ cd farmerchat-functional-automation

6 Run the Tests

Basic Run

Will prompt for tester name

$ ./run_tests.sh

With Tester Name

Provide name as argument

$ ./run_tests.sh "Your Name"

Troubleshooting

Issue Solution
No Android device connected • Ensure USB Debugging is enabled
• Check cable connection
• Run: adb kill-server && adb start-server
• Try a different USB port or cable
Maestro command not found • Add to PATH: export PATH="$PATH:$HOME/.maestro/bin"
• Re-run the Maestro installation script
Maestro fails with "java: command not found" or "Unable to find JDK" • Install Java: brew install --cask temurin@21 (macOS) or sudo apt install openjdk-21-jdk (Linux)
• Verify: java -version should print openjdk version "21..." or "17..."
• On macOS, if java still isn't found after install, restart your terminal so it picks up the new PATH
Tests timing out • Ensure stable internet connection on device
• Keep device screen unlocked during tests
• Verify FarmerChat app is working properly
System popup blocking tests • Script handles these automatically
• If stuck, manually tap "Continue installation" then "Close"
